# React Router 
![react-router image](/react/screenshots/react-router.png)

React router let us conditionally render components for Single Page application with multiple pages like a website

* Routers
    * `<BrowserRouter>`
    * `<HashRouter>`
* Route Matchers
    * `<Route>`
    * `<Switch>`
* Navigation
    * `<Link>`
    * `<NavLink>`
    * `<Redirect>`
* Hooks
    * useHistory
    * useLocation
    * useParams
